This session will guide the audience through the use of autodiff and gpu offloading in Rust.

As of a few weeks ago, we ship all needed dependencies for `std::autodiff` via rustup and nightly, so installation now takes ~5 seconds! A <a href='https://summerofcode.withgoogle.com/'>GSoC</a> contributor
is also working on adding rustup distribution for `std::offload` (gpu programming). Both these feature will be covered in this tutorial.0

<h2 style='margin-bottom:0px'>Discussion: Rust and HPC</h2>

This discussion will focus on the use of Rust on high performance computers, including discussion of
GPUs and other compute devices and parallelisation.

<h2 style='margin-bottom:0px'>Discussion: Machine learning in Rust</h2>

This discussion will focus on the use of Rust in machine learning applications.

<h2 style='margin-bottom:0px'>Discussion: Structuring Rust crates</h2>

This discussion with focus on sensible structuring of Rust crates, including discussion of how modular
we should aim to be, linting and other code tools, dependency management and version numbering.
