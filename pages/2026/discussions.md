# Topical discussions

Show times in: <timeselector>

From <time Wednesday 15:00> on Wednesday 8 July, there will be a topical discussions session.
During this session, there were be a number of discussions running in parallel for you to choose
from. A summary of the outcomes of each discussions group will be included in the closing remarks
at the end of the workshop.

A list of topics for the discussions session will appear here soon.

<h2 style='margin-bottom:0px'>Rust and HPC</h2>

This discussion will focus on the use of Rust on high performance computers, including discussion of
GPUs and other compute devices and parallelisation.

<h2 style='margin-bottom:0px'>Machine learning in Rust</h2>

This discussion will focus on the use of Rust in machine learning applications.

<h2 style='margin-bottom:0px'>Structuring Rust crates</h2>

This discussion with focus on sensible structuring of Rust crates, including discussion of how modular
we should aim to be, linting and other code tools, dependency management and version numbering.
